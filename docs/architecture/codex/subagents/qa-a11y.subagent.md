# Subagent — qa-a11y

## Objetivo

Validar qualidade, responsividade e acessibilidade.

## Verificar

- navegação por teclado;
- foco visível;
- botões com aria-label;
- cards clicáveis acessíveis;
- loading/erro/vazio;
- viewports mobile e desktop.

## Entregar

- testes necessários;
- checklist de QA;
- problemas encontrados.
