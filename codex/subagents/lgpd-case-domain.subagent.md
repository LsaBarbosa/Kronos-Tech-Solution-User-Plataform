# Subagent — LGPD Case Domain

## Objetivo
Mapear o domínio da solicitação LGPD administrativa.

## Tarefas
- Listar status possíveis.
- Listar tipos de solicitação.
- Mapear fluxo visual:
  1. Aberta
  2. Em análise
  3. Revisão do controlador
  4. Revisão legal
  5. Aprovada para exportação
  6. Concluída/parcialmente concluída
- Diferenciar solicitações exportáveis de não exportáveis.
- Garantir tratamento específico para `ANONYMIZATION` e `DELETION`.

## Saída esperada
Helpers de status, fluxo, labels, tons e próximas ações.
