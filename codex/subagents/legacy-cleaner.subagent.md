# Subagent — legacy-cleaner

## Objetivo

Remover o legado visual após a nova UI.

## Procurar

- Imports não usados.
- Cards antigos.
- Layout antigo centralizado.
- Classes antigas sem uso.
- Funções duplicadas.
- Componentes mortos.

## Saída esperada

- Lista de remoções.
- Confirmação de que só a nova implementação permanece.
