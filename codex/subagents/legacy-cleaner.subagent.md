# Subagent — Legacy Cleaner

Remova a UI antiga depois que a nova estiver funcional.

## Remover

- filtros legados;
- grid/cards antigos;
- imports mortos;
- estados mortos;
- helpers não usados.

## Preservar

- serviços compartilhados;
- tipos compartilhados;
- rota;
- guards;
- modal biométrico se usado.
