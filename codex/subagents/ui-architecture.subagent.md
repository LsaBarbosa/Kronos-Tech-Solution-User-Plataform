# Subagent — ui-architecture

## Objetivo

Definir a arquitetura de componentes para desktop e mobile.

## Desktop

- Hero.
- Painel de filtros.
- Lista de registros.
- Painel de decisão.
- Confirmações.

## Mobile

- Header compacto.
- Etapas sequenciais.
- Cards de registros.
- Bottom bar.

## Entrega

Propor estrutura de componentes e classes Tailwind sem adicionar dependências externas.
