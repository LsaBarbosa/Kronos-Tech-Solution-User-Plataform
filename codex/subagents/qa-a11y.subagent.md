# Subagent — QA & A11y

## Objetivo

Garantir qualidade, acessibilidade e responsividade.

## Tarefas

- Testar desktop e mobile.
- Garantir `aria-label` nos CTAs.
- Garantir texto junto com cor nos status.
- Garantir navegação por teclado nos cards clicáveis.
- Validar loading/error/empty.
- Rodar:
  - `npm run lint`
  - `npx tsc --noEmit`
  - `npm run build`
  - `npx vitest run`

## Viewports manuais

- Mobile: 390x844, 430x932.
- Desktop: 1366x768, 1536x864, 1920x1080.
